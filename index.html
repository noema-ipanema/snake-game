About This Project

This project explores reflexes, spatial awareness, and interactive web development through a modern Snake implementation built with plain HTML, CSS, and JavaScript. It combines responsive gameplay, lightweight performance metrics, and a polished interface to showcase both frontend fundamentals and thoughtful interaction design.
